# Edit Event

## User Story
As an authenticated member, I want to edit existing events so that I can update event information.

## Acceptance Criteria
- Authenticated users can click "Muokkaa" on event rows
- Edit form is pre-filled with current event data
- Changes can be saved with success message
- Form validation applies to edits
- Updated events reflect changes in the calendar

## Scenarios
- Auth user creates an event
- Clicks "Muokkaa" on the event
- Modifies title and submits
- Sees success message and updated event
